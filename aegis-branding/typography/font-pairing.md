# AEGIS — Typographie

## Polices

| Usage | Police | Weights | Taille indicative | Licence |
|-------|--------|---------|-------------------|---------|
| Titres H1–H3 | **Inter Tight** | 700–900 | 32–48px | [SIL Open Font](https://fonts.google.com/specimen/Inter+Tight) |
| Titres H4–H6 | **Inter** | 600–700 | 18–28px | Open (Google Fonts) |
| Corps | **Inter** | 400–500 | 14–18px | Open |
| Code / terminal | **JetBrains Mono** | 400–600 | 13–16px | [Apache 2.0](https://www.jetbrains.com/lp/mono/) |

## Paire recommandée

**Inter Tight (Bold/ExtraBold)** + **Inter (Regular)** — moderne, lisible, adaptée produit & doc technique.

### Alternative

**Poppins Bold** + **Poppins Regular** — ton légèrement plus « consumer » ; à réserver aux campagnes marketing si besoin de différenciation.

## Règles

- **Line-height** : 1.5–1.6 pour le corps ; 1.2–1.35 pour les grands titres.
- **Letter-spacing** : +0.5px à +1px sur les titres tout en caps ou petites caps.
- **Contraste** : minimum **WCAG AA** (4.5:1) pour texte normal ; viser AAA sur les CTA critiques.

## Import (web)

Voir `typography.css` (Google Fonts avec `display=swap` pour les performances).
