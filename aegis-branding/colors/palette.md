# AEGIS — Palette de couleurs

**Zokastech** · *Innovate • Connect • Grow*

| Nom | Hex | RGB | Rôle |
|-----|-----|-----|------|
| **Bleu primaire** | `#0066ff` | 0, 102, 255 | Logo, CTA, éléments clés |
| **Charcoal** | `#1a1f3a` | 26, 31, 58 | Backgrounds, texte sur fond clair inversé |
| **Orange** | `#ff6b35` | 255, 107, 53 | Performance, alertes, warnings |
| **Cyan** | `#00d9ff` | 0, 217, 255 | Réseaux, tech, surbrillance code |
| **Vert** | `#10b981` | 16, 185, 129 | Succès, validation |
| **Rouge (erreur)** | `#ef4444` | 239, 68, 68 | Erreurs (extension guidelines) |
| **Blanc cassé** | `#f8f9fa` | 248, 249, 250 | Surfaces claires |
| **Gris clair** | `#e5e7eb` | 229, 231, 235 | Bordures |
| **Gris foncé** | `#374151` | 55, 65, 81 | Texte secondaire |
| **Noir bleuté** | `#0f172a` | 15, 23, 42 | Texte principal |

## Usage

- **Primaire** : logo, boutons primaires, liens d’action.
- **Secondaire** : headers sombres, footers, cartes mode dark.
- **Accents** : états UI (warm = attention, tech = données/infra, vert = OK).
- **Neutres** : fonds, typographie, séparation.

Fichiers machine : `palette.css`, `palette.json`.
