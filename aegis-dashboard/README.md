# AEGIS Dashboard

**AEGIS by [zokastech.fr](https://zokastech.fr)** — interface d’administration (React 18, TypeScript, Vite).

## Démarrage

```bash
cd aegis-dashboard
npm install
npm run dev
```

Build production : `npm run build` (sortie dans `dist/`).
